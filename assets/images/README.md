# Images de l'app CryptoFx

Placez ici le **logo** de l'application :

- **Fichier attendu :** `logo.png`
- Format conseillé : PNG avec fond transparent (ou fond sombre pour le thème actuel).
- Taille conseillée : au moins 512×512 px pour une bonne qualité.

Le logo sera affiché sur la première page (écran d'accueil) et peut être réutilisé ailleurs dans l'app.
